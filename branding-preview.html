Brand Usage Guidelines

Logo Usage

  • Use Version 1 or 3 for primary branding
  • Use Version 6 for compact spaces (navbars)
  • Use Version 4 for app icons/favicons
  • Maintain minimum clear space around logo
  • Never distort or rotate the logo

Color Application

  • Primary gradient for CTAs and hero sections
  • Accent gradient for highlights and badges
  • Dark (#1a1a2e) for text and dark mode
  • Success green for confirmations
  • Use gradients sparingly for maximum impact

Typography

  • Headers: Inter or SF Pro Display (Bold)
  • Body: Inter or SF Pro Text (Regular)
  • Code: Fira Code or JetBrains Mono
  • Maintain hierarchy with size and weight
  • Use brand purple for accented headings

UI Components

  • Buttons: Gradient fill with white text
  • Cards: White bg with subtle shadow
  • Inputs: Light gray border, focus gradient
  • Dashboard: Dark mode with accent highlights
  • Icons: Line style with gradient strokes

Voice & Tone

  • Professional yet approachable
  • Emphasize automation and simplicity
  • Technical but not overwhelming
  • Confident without being arrogant
  • Focus on time-saving benefits

Marketing Assets

  • Use gradient backgrounds for hero sections
  • Include podcast waveform patterns
  • Showcase before/after automation
  • Use Version 5 for social media
  • AI theme with circuit/neural patterns
